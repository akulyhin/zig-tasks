Некоторые правила написания статей

Про что не писать:

не перечислять название других производителей дымоходов и торговых марок.

Про что Писать:
  • Пишем для людей, а не для поисковых систем, без воды, целевая аудитория: мужчины (40+). Дачники, владельцы частных домов, застройщики.
  • Текст должен легко читаться;
  • Читабельность. Текст должен состоять из абзацев для легкого чтения;
  • Оптимальный абзац: 3-5 строк 400-700 символов;
  • Использовать обращение к читателю - Вы (с большой буквы);
  • Использовать 2-5 подзаголовков h2. Текст должен включать в себя слова из запросов + слова для людей, выглядеть естественно и отображать суть последующего текста (примерно 50% подзаголовков не должны включать запросы, только тексты для людей).
  • Использование подзаголовков h3, если позволяет размер текста. H3 не обязательно включать слова из запросов, должні быть полезными и раскрывать более детально суть подзаголовков h2.
  • Использование название города в двух вариантах. “В Киеве” и “Киев”.
  • Равномерное размещение запросов по тексту. Не нужно помещать их всех в один абзац и повторять в одном предложении.
  • Проверять источники информации (лучше использовать 2-3 источника), чтобы не было неправдивой информации, нужен полезный контент.
  • Подобрать картинки для каждого заголовка, можно использовать наши тут.
  • Когда перечисляете и сравниваете другие типы дымоходов (кроме наших из нержавеющей стали), то нужно завершить текст таким, образом, чтобы преимущества труб из нержавеющей стали были очевидными.
